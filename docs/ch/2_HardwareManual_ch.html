    # 开箱展示

     包装盒的贴纸提供了OpenNCC的官网地址www.openncc.com.cn (opens new window)，可供下载相关的技术文档和开发包。

     打开包装盒，可以看到OpenNCC和标配的USB TPYE-C接口数据线。
